用npm安装express

发表于 2018-07-02

首先保证node和npm的安装都没问题

然后进入正题，安装express:

首先要配置环境变量，具体步骤可以百度，然后进入安装过程。
在命令行下输入：

  1. npm install express -g

  2. npm install -g express-generator

    3. 一般来说，这里就应该安装好了express.



    修改npm安装模块的全局路径和缓存路径(首先你要自己在目录下新建node_global和node_cache文件夹)：

    1. npm config set prefix “E:\nodejs\node_global”

    2. npm config set cache “E:\nodejs\node_cache”

    3. npm config get prefix检验是否更改了安装路径


    安装完以后，应该会在E:\nodejs\node_global中出现express和一个node_nodules文件夹，并且里面有东西。

    网上说检查express版本用语句express -V,可是命令行提示我没有-V 这个命令，于是我重装了好多次，仍旧没用，但使用express –version就可以检测出版本，网上的也不能全信~
